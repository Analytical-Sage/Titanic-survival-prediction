  3. Configure Git (in your terminal/command prompt):

    bash
    git config --global user.name "Your Name"

Step 2: Create a New Repository on GitHub

  1. Log in to GitHub.

  2. Click the + on the top right New repository.

  3. Fill in:

    • Repository name (e.g., used-car-price-prediction)

    • Description (optional but good)

    • Choose Public or Private

    • Click Create repository

  4. You'll get setup instructions with the repository URL copy it!

Step 3: Clone the Repository to Your Local Machine

In your terminal or Git Bash:

bash
git clone https://github.com/your-username/repository-name.git
cd repository-name

Step 4: Add Your Project Files

Add your notebooks, code files, datasets (small size only), README.md, or requirements.txt here.

Example:

  • main.ipynb

  • data/

  • README.md (write a short description of your project)

Step 5: Track Changes and Push to GitHub

bash
git add .
git commit -m "Initial commit with project files"
git push origin main

Step 6: Update Project as You Work

Repeat this cycle whenever you update something:

bash
git add .
git commit -m "Explain what you changed"
git push

Step 7: Collaborate (Optional)

  • Add collaborators under Settings > Collaborators if working with others.

  • Use Issues tab to track bugs or tasks.

  • Use Pull Requests if collaborating.

  • Keep your README.md updated.

    bash

Step 10: Share or Showcase

  • Share your repo link in your resume, LinkedIn, or portfolio.
